ARDUINO, SOFTWARE Y HARDWARE

Abril, 20 2023 Dayanne Rojas

ARDUINO:

Es importante conocer cuáles son todos y cada uno de los componentes de una placa de Arduino. Puesto que, dichos elementos son esenciales para poder empezar desde cero y así, programar tus primeros proyectos de código abierto correctamente. Por lo que, a continuación, conocerás los módulos que conforman una placa de Arduino.

SOFTWARE
El entorno de Arduino se cataloga como una IDE (Entorno de Desarrollo Integrado), lo cuál permite hacer la tarea del programador mucho más sencilla.

HARDWARE
- Microcontrolador
- Pines digitales
- Pines analógicos.
- Pines de alimentación de sensores
- Pines PWM
- Pin AREF
- Pin GND
- Botón reset
- Puerto USB
- Control de USB
- Conector de alimentación
- Regulador de voltaje
- Recepción (RX) y Transmisión (TX) Serial
- Cristal

Conceptos básicos de programación y deficinición de términos

Abril, 13 2023 Dayanne Rojas

DEFINICIÓN DE TÉRMINOS:

UN SITIO WEB: Es el conjunto de páginas web.

PÁGINA WEB: Son archivos digitales que están escritos usando HTML.

COMPONENTES

- Webhost: Es el lugar donde el sitio web está localizado físicamente
- Dirección: También conocida como URL.
- Homepage: Es la primera página que se abre cuando abrimos nuestro sitio web

¿Qué es Hosting y Dominio?

Abril, 14 2023 Dayanne Rojas

DEFINICIÓN DE TÉRMINOS:

HOSTING (Hosting / Alojamineto)
Es el espacio físico donde está almacenado el sitio Web, puede ser una computadora o un servidor.
Ejemplos:
- WebEmpresa.
- Bluehost.
- HostGator.
- PuntoPe.

DOMINIO (Cascading Style Sheets / Hojas de estilo en cascada)
Es un nombre asociado a una dirección IP física de internet, la cual determina la dirección de un sitio Web.
- La DNS es como un directorio telefónico, pero este usa direcciones Web.
- Los DNS convierten los nombres de dominio a direcciones IP.

DNS (Domain Name System)
Es el directorio telefónico de Internet. El DNS traduce los nombres de dominio a direcciones de IP para que los navegadores puedan cargar los recurso de Internet.
- Los servidores DNS suprimen la necesidad de que las perosnas memoricen extensas direcciones de IP o complejas direcciones alfanuméricas.
